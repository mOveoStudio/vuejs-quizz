# Quizz de Démo
Projet de quizz avec VueJS.

## Installer le projet et ses dépendances
```
yarn install
```

## Lancer le site en dev
**Lancer le serveur front**
```
yarn serve
```

**Lancer le serveur API**

Simple serveur php qui renvoie le contenu du fichier `server/game.json` 
```
yarn php
```

**Voir le site**

```
http://localhost:8080
```
